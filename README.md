# Games repository
This repository is a fork from https://the-french-connections.github.io/ made by @S0fian3M so I could add my own puzzles.

'My French Connections', based on 'The French Connections' is a word puzzle game similar to New York Times's Connections daily puzzles, but in French. All the puzzles in 'My French Connections' are created by myself.
